About Project

The Phishing URL Detector is a cybersecurity web application that detects whether a URL is Legitimate or Phishing in real time. It uses a hybrid machine learning and deep learning approach along with SSL certificate verification. This tool can be used by banks, enterprises, and security-conscious users to prevent phishing attacks.

Features & Workflow

Key Features
  • Hybrid prediction combining CNN, RF, and XGB models.
  • SSL certificate analysis for HTTPS websites.
  • Real-time detection with recent checks history.
  • User-friendly responsive interface.
Workflow
  1. User enters a URL.
  2. Models (CNN, RF, XGB) generate phishing probabilities.
  3. SSL certificate is verified if HTTPS is present.
  4. Final result (Phishing or Legitimate) is displayed along with SSL details.
  5. Recent checks are updated automatically.

Dataset

The system is trained on the PHIUSIIL Phishing URL Dataset, which contains URLs labeled as Legitimate (0) or Phishing (1).

  • Columns: URL, Label
  • Total Features Extracted: URL length, number of dots, hyphens, underscores, slashes, digits, HTTPS presence, suspicious TLDs.
